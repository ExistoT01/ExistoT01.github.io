C#委托（Delegate）

委托的声明

声明委托的语法
delegate <return type> <delegate-name> <parameter list>

  • parameter list : 参数列表

例如 :
public delegate int MyDelegate (string s);

该委托可被用于引用任何一个带有一个单一的 string 参数的方法，并返回一个 int 类型变量。

委托的实例化

委托对象使用 new 关键字来创建，创建时委托的参数是一个特定的函数，这个特定的函数不用写参数

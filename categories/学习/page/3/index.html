C#委托（Delegate）

委托的声明

声明委托的语法
delegate <return type> <delegate-name> <parameter list>

  • parameter list : 参数列表

例如 :
public delegate int MyDelegate (string s);

该委托可被用于引用任何一个带有一个单一的 string 参数的方法，并返回一个 int 类型变量。

C#接口

C#接口（Interface）

  • C#中的接口与抽象类十分类似，但是接口大多只是用在当只有少数方法由基类声明由派生类实现时
  • 接口本身并不实现任何功能
  • 接口定义了属性、方法和事件，这些都是接口的成员。接口只包含了成员的声明。成员的定义是派生类的责任。
  • 与抽象类一样，接口不能被实例化。
